# CSS Motion Lab

A showcase of advanced CSS animations, interactions, and theming
built with **pure CSS** and minimal JavaScript.

## Features
- Animated buttons & cards
- CSS loaders & progress bars
- Dark / Light mode
- Dynamic accent color picker
- Modern UI patterns

## What This Demonstrates
- CSS variables & theming
- Keyframe animations
- Hover & micro-interactions
- Component-based styling
- Responsive layout thinking

## Tech Stack
- HTML5
- CSS3 (Animations, Variables, Flexbox)
- Vanilla JavaScript (toggles only)
